# Практична робота № 1: Застосування звичайних Жорданових виключень

**Дисципліна:** Алгоритми систем підтримки прийняття рішень
**Виконав:** студент 3 курсу групи № 633П Коваленко М.А.

## Мета роботи
Навчитися застосовувати звичайні жорданові виключення (ЗЖВ).

## Функціонал програми
Для виконання практичної роботи було розроблено консольний застосунок мовою С#. Програма дозволяє на основі звичайних жорданових виключень реалізувати такі дії:
* Пошук оберненої матриці для довільної квадратної матриці.
* Обчислення рангу довільної прямокутної матриці.
* Розв'язання системи лінійних алгебраїчних рівнянь за допомогою оберненої матриці (1-й спосіб).
* Виведення детального згенерованого протоколу обчислень по кожному кроку ЗЖВ.

## Структура проекту
Програма розділена на два основні класи:
* **`Program`**: відповідає за керування консольним інтерфейсом та всіма обрахунками за алгоритмами.
* **`ReadHelpers`**: відповідає за обробку вхідних даних при вводі матриць.

## Вхідні дані (Варіант 12)
Програма приймає на вході кількість рядків і стовпців матриці. Для тестування використовувався Варіант 12:

**Матриця коефіцієнтів A:**
```text
-2  -1  -2
 4  -2   1
 1   3  -5
```

**Вектор вільних членів B:**
```text
 1
 5
 3
```

## Приклад результату роботи програми
На виході маємо ранг, обернену матрицю, розв’язок системи рівнянь, а також протокол обчислень:

* **Ранг вхідної матриці A:** r = 3
* **Обернена матриця:**
```text
-0,11     0,17     0,08
-0,33    -0,19     0,10
-0,22    -0,08    -0,13
```

* **Обчислення розв’язків:**
```text
X[1] = 1,00 * (-0,11) + 5,00 * (0,17) + 3,00 * (0,08)    = 1,00
X[2] = 1,00 * (-0,33) + 5,00 * (-0,19) + 3,00 * (0,10)   = -1,00
X[3] = 1,00 * (-0,22) + 5,00 * (-0,08) + 3,00 * (-0,13)  = -1,00
```
